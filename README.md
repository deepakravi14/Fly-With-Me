# Fly-With-Me-Unity3D-Game
 Simple Rocket Flying game made with Unity3D.
 Try the game at https://sharemygame.com/@DeepakRavi/fly-with-me
 Use Space bar to Thrust and Direction Keys or 'A'/'D' Key to move.
